## 下面哪一組是算術運算子？
1. **, /, ^, !, ++, --
2. *, /, %, +, - *正解*
3. &, |, +, -, /


## 以下哪個值可以與餘數運算符一起使用？
1. 3.54
2. true
3. 57 *正解*
4. "Try Me!"

> **3:** 餘數運算子僅適用於整數值
>


## 這個算式的結果是什麼？
```go
8 % 3
```
1. 4
2. 2 *正解*
3. 0
4. 1 

## 這個算式的結果是什麼？
```go
-(3 * -2)
```
1. -6
2. -1
3. 0
4. 6 *正解*


## 這個算式的結果是什麼？
```go
var degree float64 = 10 / 4
```
1. 2.5
2. 2.49
3. 2 *CORRECT*
4. 0

> **3:** 10 跟 4 都是整數值所以最後的結果會是 2
>


## 這個算式的結果是什麼？
```go
var degree float64 = 3. / 2
```
1. 1.5 *正解*
2. 1.49
3. 1
4. 0

> **1:**  `3.` 為浮點值
>
